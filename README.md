# Waves of Democracy Dashboard

A Dashboard created to visualize Democratization and Autocratization Developments using V-Dem Data.
The main goal was to show the theory of "Waves of Democratization and Autocratization" on a global scale.

The dashboard was created as the final project in the WBS Coding School Bootcamp on Data Analytics.

The Dashboard can be accessed through the Streamlit Community Cloud under this link: https://wavesofdemocracy.streamlit.app/
